## Clonar repositorio

```bash 
git clone https://github.com/YormanAlexanderCarrillo/ProyectoGradoFrontend.git
```
## Ingresar carpeta

```bash
cd ProyectoGradoFrontend
```

## Instalar paquetes necesarios

```bash
npm install
```

## Inicializar el servidor 
### Modo desarrollo

```bash
npm run dev
```
